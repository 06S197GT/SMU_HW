Climate Analysis Investigation

Summary:

The purpose behind this project was to determine whether there was a correlation between weather and latitude. To begin, we first used the OpenWeatherAPI to create a dataframe of over 500 randomized cities around the globe.
Once the dataset was assembled, we used the Python MatPlotLib library to create plots of the weather variables to show how they relate to latitude coordinates of any given location. Of the variables included were: temperature, wind speed, cloudiness and humidity.
This site was created to provide an interactive source for all of the data gathered and visualizations created.
